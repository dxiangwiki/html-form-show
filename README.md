# HTML‑Form‑Show
原生JS哈希单页路由，在线查看HTML表单元素示例以及原生源代码。

## 项目介绍
- 使用原生JavaScript实现SPA单页面哈希路由，无Vue、React等框架。
- 分为两块区域：表单预览区域（带有Tailwind美化样式）、原生HTML源码展示区。
- 源码区域自动对HTML标签进行转义，在pre标签内原样展示标签文本，并且支持一键复制原生HTML代码。
- 包含全部常用表单元素：`input(text/password/number/email/radio/checkbox/file/hidden)`、`select`、`textarea`、`button`、`fieldset`。
- 全屏页面布局，搭配渐变彩色UI界面，适配PC端。

## 使用方式
1. 将项目下载到本地；
2. 直接打开 `index.html` 文件即可运行，不需要启动服务；
3. 左侧导航栏点击对应表单分类查看预览效果与原生HTML代码。

## 项目依赖
- Tailwind‑CSS 通过CDN引入，不需要本地安装。
